# Chore Tracker

A Home Assistant integration for tracking household chores.

## Installation

### HACS (Recommended)

1. Open HACS
2. Go to "Integrations"
3. Click the three dots in the top right corner
4. Select "Custom repositories"
5. Add `https://github.com/seedybee/chore_tracker` as an integration
6. Click "Install"
7. Restart Home Assistant

### Manual Installation

1. Copy the `custom_components/chore_tracker` folder to your Home Assistant's `custom_components` directory
2. Restart Home Assistant
